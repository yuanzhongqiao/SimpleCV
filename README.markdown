<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">使用计算机视觉开源框架 SimpleCV 让计算机看到东西</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">SimpleCV 是一个开源机器视觉框架，使用 OpenCV 和 Python 编程语言。它为相机、图像处理、特征提取和格式转换提供了简洁、可读的界面。我们的使命是为普通用户提供基本机器视觉功能的综合界面，并为高级用户提供优雅的编程界面。</font></font></p>
<p dir="auto"><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">我们喜欢 SimpleCV 因为：</font></font></p>
<ul dir="auto">
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">即使是新手程序员也可以编写简单的机器视觉测试</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">相机、视频文件、图像和视频流都是可互操作的</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">可以轻松提取、排序和过滤图像特征信息</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">操作速度快，名称容易记住</font></font></li>
<li><font style="vertical-align: inherit;"><font style="vertical-align: inherit;">线性代数是严格可选的</font></font></li>
</ul>
